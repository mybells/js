## 概念

冒泡排序比较所有相邻的的两个项，如果第一个比第二个大，则交换它们。元素项向上移动至正确的顺序，就好像气泡升至表面一样。
![冒泡排序](./image/bubbleSort.gif)

## 实现

```js
function bubbleSort(arr) {
  for (let i = 1; i < arr.length; i++) {
    for (let j = 0; j < arr.length - i; j++) {
      if (arr[j] > arr[j + 1]) {
        ;[arr[j], arr[j + 1]] = [arr[j + 1], arr[j]]
      }
    }
  }
}
```
## 复杂度
时间复杂度：O(n2)

空间复杂度:O(1)
