sticky定位，须指定 top, right, bottom 或 left 四个阈值其中之一，才可使粘性定位生效。否则其行为与相对定位相同。
position:fixed;和position:absolute;这两种定位时，没有top/bottom时相对于父元素上下定位，没有left/right时相对于父元素左右定位，可用margin定位。 如果有top/bottom：postion:fixed;相对于浏览器视口上下定位。如果有left/right：fixed相对于浏览器视口左右定位。
如果有top/bottom：postion:absolute;相对于最近的非 static 祖先元素上下定位(啥都不写也是static默认定位)。如果有left/right：fixed相对于最近的非 static 祖先元素左右定位。
